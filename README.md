Yet another Brainfuck interpreter in Ruby. 

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'yarbf'
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install yarbf
```

## Usage

After installation, execute:

```shell
$ yarbf --help
```

It will show its usage information.
